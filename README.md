# Guide Linux – Laboratoires

Ce dépôt contient le guide pour mettre en place un environnement Linux pour les travaux pratiques d’administration système.

## Contenu
- Installation et utilisation de **WSL** sous Windows  
- Déploiement de Linux avec **VirtualBox** et import d’une VM pédagogique  
- Utilisation de Linux en ligne avec **Terminal Tutor** et **KillerCoda**  
- Pratique Linux avec **Docker** et plateformes pédagogiques comme **Linux Journey**  
- Défis interactifs avec **OverTheWire – Bandit**
